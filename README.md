# 소개
무선 LAN 및 Bluetooth 연결이 가능한 단일 보드 컴퓨터인 Raspberry Pi 3b와 온습도 센서인 dht22을 연결

Raspberry Pi에 Alexa 서비스를 다운

AWS에서 Skill Add 조건에 대한 입력 값을 Alexa에 말함

Alexa는 해당 Skill에 따른 대답
